# fenio/scoop-bucket

A [Scoop](https://scoop.sh) bucket for [Zod](https://github.com/fenio/zod) — a
modernized SDL2 build of the Zod Engine (open-source remake of the 1996 RTS *Z*).

```powershell
scoop bucket add fenio https://github.com/fenio/scoop-bucket
scoop install zod
zod
```

> **Note:** the Windows build is **experimental and not yet runtime-tested** —
> it compiles and links in CI, but the game has not been verified running on
> Windows. Feedback welcome.
